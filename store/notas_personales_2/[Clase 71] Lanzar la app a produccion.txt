En este curso hemos utilizado vercel como hosting de la aplicacion.

Si necesitaramos que angular comprima el archivo a html, css y js para produccion manual podriamos utilizar

ng build

Pero como nosotros estamos utilizando vercel con el comando:

vercel

y luego

vercel --prod
